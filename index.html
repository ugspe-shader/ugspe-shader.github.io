Home

Unreal Graphic Shader or UGSPE is a Shader for Minecraft PE. This Shader will enhanced The Default Color of Minecraft to more Colorful and Realistic. UGSPE has 2 pack. First, Unreal Graphic Shader. Second, Unreal Graphic Shader : Default++.

UGSPE 2.0

Unreal graphic Shader with Realistic Coloring. Have some feature like Volumetric Cloud, Realistic Water, and Many more. This Shader has 4 variant : High, Medium, Low, and Lite. So this shader can be run on all phones, except phone with OpenGL 2.0.

Features
  • Tone Coloring
  • Vibrant Gradient
  • Waving Leaves and Grass
  • Volumetric Cloud
  • Realistic Water
  • Caustic
  • Sun Rays
  • Fog

UGSPE : Default++

The Default Minecraft Shader with More Improvement like Fog, Saturation, Cloud. This Shader has 2 variants. First, With Fake Reflection. Second, Without Fake Reflection

Features
  • Improved Fog
  • Waving Leaves, Grass, and Water
  • Improved Saturation
  • Improved Cloud
  • Fresnel
  • Fake Reflection on Water
  • Caustic
